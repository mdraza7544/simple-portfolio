Simple Portfolio
A clean and modern personal portfolio website built with HTML, CSS, and JavaScript.

Features
•	Responsive Design: Works seamlessly across all devices
•	Clean UI: Modern interface with smooth animations
•	Easy to Customize: Well-organized code structure
•	Lightweight: Fast loading and optimized

Technologies Used
•	HTML5 (51.3%)
•	CSS3 (40.0%)
•	JavaScript (8.7%)

Project Structure
Simple-Portfolio/
├── index.html
├── style.css
├── script.js
└── .gitignore

Getting Started
Clone the repository
git clone https://github.com/mdraza7544/Simple-Portfolio.git
cd Simple-Portfolio

Open the project
Simply open index.html in your browser or use:
open index.html

Customization
•	Edit index.html for personal information
•	Modify style.css for colors, fonts, and layout
•	Enhance script.js for additional functionality
